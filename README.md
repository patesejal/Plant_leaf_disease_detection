# Plant Leaf Disease Detection

A deep learning-based image classification project designed to detect plant leaf diseases using Convolutional Neural Networks (CNNs).

## Overview
This project uses Python and TensorFlow to classify diseased and healthy plant leaves across multiple categories using image preprocessing and deep learning workflows.

## Technologies Used
- Python
- TensorFlow
- CNN
- NumPy
- Pandas
- Jupyter Notebook

## Features
- Image preprocessing
- Data augmentation
- CNN model training
- Disease classification
- Accuracy evaluation

## Results
Achieved 95% classification accuracy across multiple plant disease categories.
